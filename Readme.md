## 项目介绍

本项目是由**我个人独立**、使用 React 复刻 Vue-Vben-Admin，**还没复刻完**，供大家学习，大家有意见可以提出来

技术栈：`React18 + Zustand + Typescript + Pnpm + TailWindcss + Vite`

```bash
# 安装依赖
pnpm i
# 运行项目
pnpm run sunshine:dev
# 构建项目
pnpm run sunshine:build
```
